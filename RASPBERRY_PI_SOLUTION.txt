PATH 3: FTDI CHIP ANALYSIS (HARDWARE APPROACH)
===============================================

Your Nisscom device uses FTDI USB-to-serial chip. We can:

1. Use ftdi_sio driver on Linux
2. Send FTDI-specific commands to configure pins
3. Some K-line adapters use FTDI pins for special functions

Investigation steps:
  - Check which FTDI chip (FT232, FT2232, etc.)
  - Try FTDI bitbang mode
  - Configure special pins (CBUS pins might control K-line)

This requires:
  - libftdi on Raspberry Pi
  - Python pylibftdi
  - Trial and error with pin configurations
