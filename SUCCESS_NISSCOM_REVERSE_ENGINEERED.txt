PROTOCOL DETAILS DISCOVERED
================================================================================

CONNECTION:
  Port: COM5 (USB Serial - FTDI)
  Baudrate: 38400
  Data bits: 8
  Parity: None
  Stop bits: 1

INITIALIZATION SEQUENCE:
  1. Send: 0x00          (Sync/Wakeup)
  2. Send: 0x81          (Init Command)
  3. Send: 0x10 0xFC 0x81 (Handshake Part 1)
  4. Send: 0x0E          (Handshake Part 2)

COMMAND FORMATS:

  Type 2 (4-byte format):
    [0x02] [Param1] [Param2] [Checksum_SUM]

    Example: 02 1A 81 9D
      0x02 = Command type
      0x1A = Parameter 1
      0x81 = Parameter 2
      0x9D = Checksum (sum of first 3 bytes)

  Type 5 (7-byte format) - Register/Memory Read:
    [0x05] [0x22] [Register] [Address] [Length] [Flags] [Checksum_SUM]

    Example: 05 22 11 00 04 01 3D
      0x05 = Command type
      0x22 = Subcommand
      0x11 = Register number
      0x00 = Address offset
      0x04 = Number of bytes to read
      0x01 = Flags
      0x3D = Checksum (sum of first 6 bytes)

CHECKSUM ALGORITHM:
  Type: SUM (not XOR!)
  Method: Add all bytes (except checksum), take lower 8 bits
  Formula: checksum = sum(bytes) & 0xFF
