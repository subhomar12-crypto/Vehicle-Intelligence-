WHAT TO LOOK FOR (Preview)
================================================================================

In the capture, we're looking for packets like:

  URB_CONTROL out
    bmRequestType: 0x40 (Vendor-specific)
    bRequest: 0x??
    wValue: 0x????
    wIndex: 0x????
    Data: [bytes]

These are the commands that activate the device!

Common FTDI vendor commands:
  - 0x00: Reset
  - 0x01: Modem control
  - 0x02: Set flow control
  - 0x03: Set baud rate
  - 0x04: Set data characteristics
  - 0x09: Set latency timer
  - Custom commands for K-line activation
