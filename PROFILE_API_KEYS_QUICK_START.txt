HOW TO GENERATE A PROFILE-BASED API KEY:
-----------------------------------------

1. Open Predict Desktop App
2. Go to "Server" tab
3. Click "➕ Generate New Key"
4. SELECT A VEHICLE PROFILE from dropdown
   - Example: "Omar (Nissan Patrol Y61 2003)"
5. Enter API Key Name
   - Example: "Android Phone #1"
6. Click "Generate"
7. COPY THE KEY (shown only once!)
8. Configure your Android app:
   - API Key: [paste the key]
   - Server IP: [your computer's IP]
   - Port: 8000

HOW TO DELETE AN API KEY:
--------------------------
1. Click "Delete" button next to the key
2. Confirm deletion in the popup dialog
3. Key is permanently removed
4. ⚠️ WARNING: Android devices using this key will lose access

SECURITY BENEFITS:
------------------
✓ Each vehicle has its own unique API key
✓ No shared keys across vehicles
✓ Easy to revoke individual keys
✓ Clear ownership and audit trail

IMPORTANT NOTES:
----------------
• Old API keys (without profile) will show "N/A" in Profile column
• Recommended: Regenerate all old keys to link them to profiles
• Each Android device needs a separate API key per vehicle
• Keys cannot be retrieved after generation (stored as hashes)

TROUBLESHOOTING:
----------------
Issue: "No vehicle profiles found"
→ Create a vehicle profile first in Profiles tab

Issue: Profile shows "N/A"
→ Delete old key and generate new profile-based key

Issue: Cannot see profiles in dropdown
→ Check database: C:\D Drive\Predict\data\vehicle_profiles.db
