| Data Type | Description | Frequency |
|-----------|-------------|-----------|

| Throttle Position | Accelerator pedal position | 1-5 seconds |
| Intake Air Temperature | Air entering engine | 10 seconds |

| Engine Load | Current engine load percentage | 5 seconds |
| Fuel System Status | Open/closed loop status | 10 seconds |
| Oxygen Sensor Data | Emission sensor readings | 10 seconds |
| Battery Voltage | Vehicle electrical voltage | 30 seconds |
| Diagnostic Trouble Codes (DTCs) | Error codes stored in ECU | On request |
| Freeze Frame Data | Conditions when DTC set | On request |
| Emissions Readiness | I/M monitor status | On request |
